> TwitchTetris --help

Use the keyboard controls to rotate and move the blocks as they fall

Place the blocks to form horizontal lines, which will be removed and make all the blocks above fall down. If the blocks reach the top of the screen, the game is over!

Clear multiple lines at a time, or clear lines in consective moves to earn more points.
> man TwitchTeteris

TwitchTetris is an open-source implementation of Tetris, the classic falling-block game that we all know and love, designed for those who demand the best performace from their tetris game.

TwitchTetris aims to be the fastest implementation of Tetris to be played within a browser. This is possible because it's implemented completely in Html5/JavaScript. It was developed out of the sad state of other available Html5 Tetris games. TwitchTetris should have the fastest reaction time to user input of any complete Tetris implementation, and should perform the most consistently over all modern browsers.

This project is a fork of TwitchTetris. This fork aims to make the best Tetris browser game that can be played without a website.

Ads may be displayed in the future (Not Google Ads! We respect your privacy).
